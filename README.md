# 🔍 Universal OCR Pro

A powerful browser-based OCR (Optical Character Recognition) application that extracts text from images, PDFs, and documents with real-time translation and text-to-speech capabilities.

## ✨ Features

- 📁 **Multi-Format Support** - Upload PDF, Images (PNG, JPG, JPEG, BMP, TIFF), DOCX, and TXT files
- 🌐 **Real-time Translation** - Translate text to 12+ languages including Hindi, French, German, Japanese, Korean, Chinese
- 🔊 **Text-to-Speech** - Listen to translated text with multiple voice options and adjustable speed
- 📊 **Side-by-Side View** - Compare original extracted text with translated version
- 📥 **Download Results** - Save both extracted and translated text as .txt files
- 🎨 **Modern UI** - Clean, responsive design with dark theme
- ⚡ **Fast Processing** - Client-side processing, no server required

## 📋 How to Use

1. **Upload a file** - Click or drag & drop any supported file
2. **Select language** - Choose your target translation language
3. **Pick voice** - Select accent and speech speed
4. **Process** - Click "Process Document" button
5. **Review results** - See extracted and translated text
6. **Listen** - Click "Play Audio" to hear the translation
7. **Download** - Save results for later use

## 📂 Supported File Types

| Type | Extensions | Extraction Method |
|------|------------|-------------------|
| PDF | .pdf | Text extraction with PDF.js |
| Images | .png, .jpg, .jpeg, .bmp, .tiff | OCR with Tesseract.js |
| Word | .docx | Text extraction with Mammoth.js |
| Text | .txt | Direct reading |

## 🛠️ Technologies Used

- **Tesseract.js** - OCR engine for image text extraction
- **PDF.js** - PDF text extraction
- **Mammoth.js** - DOCX document parsing
- **Google Translate API** - Real-time translation
- **Web Speech API** - Text-to-speech synthesis
- **HTML5/CSS3** - Modern responsive UI
